Developed a full-stack web application using Java, Spring Boot, React.js, and SQL Server (SSMS) for efficient URL shortening and analytics tracking. Integrated Postman for API testing and validation. The application enables users to create shortened URLs, view click analytics (including location and traffic sources), and manage URLs securely. Focused on performance, data integrity, and a seamless user experience.

Key Features:

Custom URL shortening with secure redirection

Real-time analytics dashboard to track link performance

Reactive and user-friendly interface built with React.js

RESTful APIs developed using Spring Boot and tested with Postman

Data storage and management using SQL Server

This repository contains the fronted as well as backend.
